# Karma Yoga for Maker

Ein MakerSpace lebt von seinen Menschen. Von denen, die nach dem Lasern den Tisch abwischen, die Absaugung warten, eine Einweisung geben oder einfach mal den Mull rausbringen. All das passiert jeden Tag - oft unbemerkt, selten gewurdigt.

**Karma Yoga for Maker** macht diese stillen Beitrage sichtbar. Mitglieder posten Wunsche an die Community - "Die Absaugung braucht neue Sacke", "Werkstatt aufraumen", "Wer macht Aufsicht?" - und andere erfullen sie. Dafur gibt es Karma Punkte. An wiederkehrenden Stationen wie der Absaugung oder der Kuche hangt ein QR-Code - einmal scannen, erledigt markieren, Karma kassieren.

Im Karma Board sieht jeder wer sich einbringt. Nicht um zu bewerten, sondern um zu zeigen: Es gibt Menschen die sich kummern. Und das ist grossartig.

Jeder Wunsch ist im Hintergrund ein GitHub Issue, jede Erledigung ein Kommentar. Alles transparent, nachvollziehbar und dokumentiert - ohne dass die Nutzer davon etwas mitbekommen. Fur sie ist es einfach eine schone App.
